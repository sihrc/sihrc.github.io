Getting Started in Android™

Android Studio

First, let's download the Integrated Development Environment (IDE) for this course, Android Studio.

Android Studio is a fairly new IDE branched off from Intellij IDEA. It comes with several tools to help us easily create and deploy Android projects. Some of these tools include the SDK Manager, Gradle, Layout Editer, and VCS support. We will be using these throughout the course.

SDK Manager

Once you've gotten Android Studio running, we'll need to install the development kits necessary for Android development. Take a look here and grab the latest version.

Hello World

Let's get started with a simple Hello World. We won't be using much java this time around to give those who are unfamiliar with Java a chance to get more comfortable.

We'll be working with XML for this tutorial. Android uses XML for static layout resources. These layout resources determine content, layout, and styles for the user interface of your Android application. Although you can change the content/layout/styles dynamically in the Java code, having statically defined resources allows user interface changes without having to recompile the java code (this saves a lot of time in the long run when dealing with minor UI tweaks).

Creating the Project

We will need to create a new project using the Gradle Build System that comes with Android Studio. Gradle will make your life easier when it comes to building and managing an Android application.
